YouTube Search doesn't know
when you're exploring

When you search for something you want to actually understand, YouTube gives you the same ranked list it always does. No structure, no starting point, no sense of the bigger picture.

These are two prototypes for a better experience on research and exploratory queries, using Gemini to organize creator content in a way that helps you go deep.

Query type 1
Exploratory
Pure research queries
Big topics with a lot of ground to cover. Users need a map, not a list.
Query type 2
Hybrid
Informational + exploratory
How-to questions where the answer is also a whole landscape to navigate.

The thinking behind this
🎯
The problem
YouTube search treats every query the same way. But when someone is trying to learn something, a ranked list of videos isn't that useful. You don't know where to start or how the pieces connect.
🤖
Why Gemini, why now
Gemini can read across thousands of videos, figure out what a query is really asking for, and organize results in a way that actually helps. It's not replacing creators, it's making their content easier to navigate.
🗺
The UI adapts to intent
Exploratory queries get a topic map and a learning path. Hybrid queries get an action plan. The design changes based on what you're trying to do, not a single template applied to everything.
📐
What stays the same
The regular video grid is still there. The new layer sits on top and users can scroll past it if they want. It's additive, not a replacement for how search already works.
